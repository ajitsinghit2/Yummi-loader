Welcome to GitHub Pages.

What's this? Yummi-loading it's nothing more than a simple LESS file with some CSS3 animations. Using them is straighforward, just add 'off' class on your body element and change it with 'on' on $(window).load() or $(document).ready() events. Try it now.

<body class="off">
  <div class="fadein scaleInv anim_1">
    <p class="fadein scaleInv anim_2">
      <a class="fadein scale anim_n"></a>
    </p>
  </div>
</body>

Here we toggle on/off class on $(window).load() event, that's your choice.

$(window).load(function() {
  $('body').toggleClass('on off');
});

Configuring LESS file is easy, just choose the number of elements you are working on, the delay between them and your are good.

@iterations:	15;	// number of elements to apply effects on
@delay:		0.15;	// delay between elements transitions

There are some other options you can set, that's up to you. 

@opacity_start: 		0;
@opacity_finish:		1;
@opacity_speed_in:		0.5s; 	// seconds
@opacity_speed_out:		0.5s; 	// seconds
@opacity_delay_in:		0.5s; 	// seconds
@opacity_delay_out:		0.5s; 	// seconds
@opacity_easing_in: 		cubic-bezier(0,0,1,1);
@opacity_easing_out:		cubic-bezier(0,0,1,1);
